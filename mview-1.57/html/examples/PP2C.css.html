Blocked letters¶

A hand finished CLUSTAL alignment using built-in colour palettes and consensus pattern residue equivalence classes: member residues of the 80% consensus class for each column are coloured according to their own property.

The -css on option produces the blocked letter effect.

Command line¶

mview -in plain -width 130 -ruler on -coloring consensus -threshold 80 -consensus on -con_coloring any -html head -css on PP2C.txt
